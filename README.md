# scrape
Simple Image Web Scraper


```
usage: scrape.py [-h] [--delimiter DELIMITER] [--csv CSV | --url URL [URL ...]]

Web Image Scraper

optional arguments:
  -h, --help            show this help message and exit
  --delimiter DELIMITER, -d DELIMITER
                        Change CSV Delimiter. TAB is default.
  --list, -l            Only List Found Image Paths
  --csv CSV, -c CSV     Import URL List. NOTE: Needs Column 'url'
  --url URL [URL ...], -u URL [URL ...]
                        Scrape One or More URLs
                     
                     
                     
```
