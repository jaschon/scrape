# scrape
Simple Image Web Scraper


```
usage: scrape.py [-h] [--delimiter DELIMITER] [--csv CSV | --url URL [URL ...]]

Web Image Scraper

optional arguments:
  -h, --help            show this help message and exit
  --delimiter DELIMITER, -d DELIMITER
                        Change CSV Delimiter. TAB is default.
  --csv CSV, -c CSV, --list CSV, -l CSV
                        Import Website List. NOTE: Needs Columns (first_name, last_name, website)
  --url URL [URL ...], -u URL [URL ...], --website URL [URL ...], -w URL [URL ...]
                        Scrape One or More Websites
                     
                     
                     
```
